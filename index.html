Browser Security Limitation:

Web browsers do not allow websites to directly capture your system's speaker output for privacy and security reasons.

To capture desktop audio, please add a Display Capture source instead, and ensure you check the box that says "Share system audio" or "Also share tab audio" when the browser prompt appears.
